Iconos de Google font

En la cabecera de documento html insertamos una etiqueta link con el servidor de los iconos, tal que asi :

<link rel="stylesheet" href="https://fonts.googleapis.com/icon?family=Material+Icons">

Inserccion de ICONOS de Google font

Para insertar iconos para que se vean en nuestra pagina utilizaremos el siguiente codigo de elementosd HTML <i> o <span>

<i class="material-icons"> home</i>

home

Si queremos podemos combinar etiquetas con el <i> o el <span>

Simplemente insertando etiquetas dentro de otras como a continuacion :

<a href="www.google.com"><i class="material-icons"> home</i></a>

home
