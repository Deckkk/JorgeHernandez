Iconos de Font awesome 5

En la cabecera de documento html insertamos una etiqueta script con el servidor de los iconos, tal que asi :

<script src="https://kit.fontawesome.com/74ff1e73d3.js"><script>

Teniendo en cuenta que para Font awesome 5 icons necesitaremos un link personalizado que nos facilitaran al registrarnos en la pagina Web de Font awesome

Pagina oficial de font awesome

Inserccion de ICONOS de Google font

Para insertar iconos para que se vean en nuestra pagina utilizaremos el siguiente codigo de elementosd HTML <i> o <span>

<i class="fas fa-home"></i>

Si queremos podemos combinar etiquetas con el <i> o el <span>

Simplemente insertando etiquetas dentro de otras como a continuacion :

<a href="www.google.com"><i class="fas fa-home"></i></a>

Con una hoja de estilos CSS

En el propio documento HTML
