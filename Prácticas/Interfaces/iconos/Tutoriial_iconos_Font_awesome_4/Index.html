Iconos de Font Awesome 4

En la cabecera de documento html insertamos una etiqueta link con el servidor de los iconos, tal que asi :

<link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/4.7.0/css/font-awesome.min.css">

Inserccion de ICONOS de Font Awesome 4

Para insertar iconos para que se vean en nuestra pagina utilizaremos el siguiente codigo de elementos HTML <i> o <span>

<i class="fa fa-home"></i>

Si queremos podemos combinar etiquetas con el <i> o el <span>

Simplemente insertando etiquetas dentro de otras como a continuacion :

<a href="www.google.com"><i class="fa fa-home"></i></a>
